Central Processor (CPU)

Lithography (nm): The physical size of transistors. Smaller means more power in less space.
Clock Cycles: The rhythm at which the CPU executes instructions (measured in GHz).
Hyper-Threading: Virtualizing a single core into two to handle parallel tasks efficiently.
Cache (L1/L2/L3): Ultra-fast memory located directly on the chip to reduce latency.
← Back to Hardware

Graphics Unit (GPU)

Parallel Processing: Unlike CPUs, GPUs use thousands of cores to calculate pixels simultaneously.
VRAM (GDDR6X): Specialized memory with massive bandwidth for textures and 3D geometry.
Ray Tracing: Hardware acceleration for calculating real-time light paths and reflections.
← Back to Hardware

Memory (RAM)

Volatile Storage: Data is only retained while electricity flows; power off means data loss.
Dual Channel: Using two sticks doubles the communication bandwidth to the CPU.
Timings (CL): CAS Latency determines the delay between requesting data and receiving it.
← Back to Hardware

NVMe Storage

NAND Flash: Uses floating-gate transistors to trap electrons, storing data without power.
PCIe Bus: Connects directly to the CPU lanes, bypassing the slower SATA interface.
← Back to Hardware

Thermal Systems

AIO Liquid: All-In-One loops use a pump and radiator to move heat away from the chip.
Heat Pipes: Copper tubes with phase-change liquid that vaporizes to carry heat efficiently.
← Back to Software

Operating System

The Kernel: The absolute core program that has complete control over everything in the system.
Memory Paging: Swapping inactive RAM data to the SSD when memory runs out.
Interrupts: Hardware signals (like a mouse click) that tell the CPU to pause and pay attention.
← Back to Software

Artificial Intelligence

Neural Networks: Layers of algorithms designed to mimic human brain neuron connections.
Inference: The process of a trained model making a prediction based on new data.
LLMs: Large Language Models that predict the next logical word in a sequence.
← Back to Software

Cybersecurity

Encryption (AES-256): Scrambling data so mathematically complex it would take billions of years to crack.
Handshakes: The secure negotiation between client and server to establish trust.
Zero Day: A vulnerability known to hackers but not yet fixed by the developers.
← Back to Software

Cloud Computing

Virtualization: Running multiple 'virtual' computers on one single physical server.
Scalability: Automatically adding more servers during high traffic spikes.
SaaS: Software as a Service - using apps via browser rather than installing them.
